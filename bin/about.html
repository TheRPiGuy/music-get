About music-get

One of the (many) music server implementations for UWCS.

While the other options out there may be shinier, music-get is intended to be an easily maintainable alternative prioritising simplicity over extra features. Made by William Seymour (mcnutty).

The source for music-get is available on github. Feel free to submit a pull request if you spot a bug or want to implement a feature :)
